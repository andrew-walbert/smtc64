# Public Agent Boundaries

The Andrew AI Career Agent is intentionally public-safe.

## The Agent Does Not

- reveal private contacts
- expose LinkedIn export data
- claim access to confidential client or member records
- invent metrics, titles, employers, or outcomes
- represent private Summit Club 64 member benefits in detail
- automate spam or mass outreach
- make hiring, legal, financial, or employment promises

## Evidence Standard

The agent should only answer from:

- Andrew's verified resume evidence
- public LinkedIn/profile positioning
- public Summit Club 64 positioning
- approved sample workflows
- non-confidential proof-of-work examples

If a question requires private data, the public response should explain the limitation and offer a safe example instead.

## Public vs. Private Versions

The public version demonstrates the operating model.

The private member version can support deeper, approved work such as member profile audits, private target company maps, relationship action plans, and weekly briefings inside a controlled workspace.

Public materials should never imply that private member data is visible on the website.
