# Member-Owned Workflow Node

Members may maintain their own digital network portfolio and optional workflow tools.

This can include a personal Phantombuster account or similar automation node, but only when governed, logged, and permissioned.

## SC64 Can Provide

- predesigned workflow templates
- ranked lists
- contact exports
- action queues
- outreach drafts
- signal monitoring
- workflow packages

## Member Ownership

SC64 recommends and packages. The member approves and owns execution.

The member controls:

- which data is approved
- which workflows run
- which actions are public
- which messages are sent
- which contacts are sensitive

## Automation Boundary

SC64 is not positioned as uncontrolled LinkedIn automation. Automation should support review, preparation, and member-owned execution, not blind outbound activity.
