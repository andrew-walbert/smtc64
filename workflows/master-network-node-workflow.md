# Master Network Node Workflow

The Master Network Node is the trusted private layer behind the SC64 Member Network Agent.

## Trusted Inputs

- LinkedIn Sales Navigator
- member-approved companies
- member-approved people
- trusted groups
- CRM / Airtable / Google Sheets
- approved signal ingestion tools
- Telegram / Slack / email delivery
- human review

## Process

1. Load approved member boundaries
2. Search the trusted private layer first
3. Rank records by relevance, relationship path, trust level, and sensitivity
4. Mark unverified records as needs review
5. Prepare recommendations or review queues
6. Escalate sensitive situations
7. Add records to the trusted pool only after review

## Principle

The Master Network Node is not a public database and not an uncontrolled scraping engine. It is a permissioned trust layer for member-owned network intelligence.
