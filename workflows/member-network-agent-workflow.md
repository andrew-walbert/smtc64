# Member Network Agent Workflow

The Member Network Agent helps an approved member stay connected to the companies, people, opportunities, tools, groups, recruiters, peer networks, and signals where mutual value is most likely.

## Workflow

1. Confirm member boundaries and approved market
2. Monitor approved companies, people, groups, tools, recruiters, and roles
3. Filter noise from useful signal
4. Rank relevant opportunities, people, and warm paths
5. Recommend only when action is warranted
6. Draft language in the member's voice when requested
7. Package review queues for approval
8. Track outcomes and update the trusted pool

## Member Questions

- What is happening in my network that I should know about?
- Who should I reconnect with discreetly this week?
- Which approved companies are hiring quietly?
- Which opportunities are worth attention without signaling I am looking?

## Guardrail

The agent may monitor, recommend, draft, and package. It should not engage publicly without explicit approval.
